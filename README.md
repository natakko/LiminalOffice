# Liminal Office: Prototype

Прототип игрового уровня в жанре Liminal Spaces на Unity, сфокусированный на создание атмосферы через интерактивное окружение и аномальные механики.
Основной геймплей реализован: аномалии, освещение, звук. В планах — расширение библиотеки ассетов и доработка трёх префабов (CreepyOffice, EXIT, NoWayOut).

## Описание проекта

Проект представляет собой реализацию офисного пространства с элементами хоррора и лиминальности. Основное внимание уделено работе с освещением, звуковым дизайном и скриптовыми сценариями, которые создают ощущение неестественности происходящего.

## Техническая реализация

### 3D Моделирование и пропсы
1. **Создание ассетов в Blender** — разработан пак Workstation (стол, монитор, клавиатура, мышь, стул).
2. **Физика объектов** — настроено физическое взаимодействие для элементов рабочего места (клавиатура, мышь, стул).
3. **Организация сцены** — выстроена четкая иерархия объектов, созданы префабы для коридоров, кабинетов и элементов декора.

### Скрипты и механики
1. **Интерактивные двери** — логика открывания и взаимодействия игрока с окружением.
2. **Пространственные аномалии** — реализация исчезающих стен и системы «одностороннего прохода» (вход без возможности возврата).
3. **Визуальные аномалии** — скрипты мерцания света и динамического появления/исчезновения надписей на стенах.
4. **Звуковой дизайн** — система триггеров для воспроизведения жутких звуков и позиционирование эмбиента.

### Визуал и анимация
1. **Анимации** — настроены переходы для дверей и аномальных объектов.
2. **Эффекты окружения** — использование локального тумана (Fog) и специфических материалов для усиления атмосферы.

## Текущий статус и планы
В настоящий момент ведется активная доработка следующих префабов:
* **CreepyOffice**
* **EXIT**
* **NoWayOut**

Также планируется расширение библиотеки контента новыми моделями, созданными в Blender.
